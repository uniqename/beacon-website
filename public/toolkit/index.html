This toolkit gives any organization the structures, forms, templates, and guides to replicate Beacon's survivor support model in 3–4 months. Start with the How to Use This Toolkit guide, then follow the 90-Day Action Plan. Every form is fillable on-screen and printable as a PDF (use Cmd+P / Ctrl+P). To add new forms, copy any existing file and update the content — the shared stylesheet handles all formatting.
